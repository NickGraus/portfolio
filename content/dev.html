CSS

Flexbox

Om de css flexbox onder de knie te krijgen heb ik verschillende tools gebruikt om mij verder te helpen met dit concept. De tools die hiervoor gebruikt heb zijn Flexbox Froggy en Flexbox Defense. Tijdens het gebruiken van deze tools is voor mij duidelijk geworden wat de mogelijkheden van een flexbox zijn. Alvorens heb ik nog de educatieve video’s op Canvas bekeken die betrekking hebben op de flexbox. Nadat ik bovenstaande tools had gebruikt, heb ik zelf een standaard website proberen te bouwen waarbij ik flexbox gebruik voor de uitlijning van verschillende div’s.

Door het bekijken van de video’s en het gebruiken van de tools die op internet stonden heb ik een veel beter beeld gekregen over de toepassingsmogelijkheden van de flexbox. Daarnaast heb ik geleerd dat het beter is om vaker dan een tot twee keer te oefenen met een bepaald concept. Alhoewel ik het concept van een flexbox wel begreep in week 1, heb ik in deze week geleerd hoe efficiënt een flexbox kan zijn.

Door in de toekomst gebruik te maken van de CSS flexbox kan ik een stuk efficienter een website bouwen en heb ik veel minder tijd nodig om te positioneren. Hierdoor ben ik een betere media vormgever geworden.

Grid

Nadat ik geleerd heb over de functionaliteiten van de flexbox, heb ik mij beziggehouden bij DEV met de CSS-grid lay-out, transitions, filters en CSS-animaties. Om de grid-layout te leren heb ik gekeken naar het verschil tussen de flexbox en een grid-layout. Hierbij ben ik erachter gekomen dat je met flex in een enkele dimensie (1D) de lay-out kan plaatsen en dat je met grid in een dubbele dimensie (2D) kan werken. Daarnaast heb ik de verschillende mogelijkheden geleerd van transitions, welke filters er zijn en hoe animeren werk. Om grid te leren toepassen heb ik opnieuw een standaard website gebouwd zoals ik bij de flexbox ook heb gedaan, alleen heb ik deze keer grid gebruikt voor de layout van de website.

Door het toepassen van verschillende transitions, filters en animaties kan ik makkelijk leuke effecten toepassen zonder dat ik hier javascript voor hoef te gebruiken. Door een grid te gebruiken maakt het indelen van de lay-out een stuk makkelijker. Hierdoor blijft de code duidelijker.
